# MyloAir 测试用例 (完整版)

## 1. 应用启动与安全 (Security & Launch)

- [x] **TC-001: 初始启动**: 应用在 <3秒 内启动，加载无奔溃。
- [ ] **TC-002: 主密码设置**: 首次启动引导设置主密码。
- [ ] **TC-003: 解锁验证**: 输入正确主密码解锁；错误密码提示错误。
- [ ] **TC-004: 自动锁定**: 闲置 5 分钟（或设置的时间）后自动锁定应用 (需求 4.2.2)。

## 2. 密码管理 (Password Management)

- [x] **TC-006: 密码列表显示**: 列表正常加载。
- [x] **TC-007: 用户名脱敏**: 验证列表中的用户名符合脱敏规则 (需求 2.1.5)：
  - 邮箱: `quanss168@yonyon.com` -> `qua**@yonyon.com`
  - 手机: `13122024777` -> `131****777`
  - 普通: `admin` -> `ad**n`
- [x] **TC-008: 隐私切换**: 点击“眼睛”图标，密码/敏感信息在明文与密文间切换 (需求 2.1.1)。
- [x] **TC-009: 复制功能**: 点击用户名/密码可复制完整内容到剪贴板。
- [x] **TC-010: 添加密码**: 完整填写(标题, 用户名, 密码, URL, 备注, 分组)并保存。
- [x] **TC-011: 编辑密码**: 修改现有记录，保存后生效。
- [x] **TC-012: 密码历史**: 修改密码后，在“历史记录”中能看到旧密码 (需求 2.1.1)。
- [x] **TC-013: 删除密码**: 删除条目，确认从列表消失。

## 3. 分组管理 (Group Management)

- [x] **TC-014: 分组树展示**: 左侧树形结构正确显示，支持展开/折叠。
- [x] **TC-015: 新建分组**: 创建新分组，支持选择颜色 (10种颜色) (需求 2.1.3)。
- [ ] **TC-016: 唯一性检查**: 尝试创建同名分组，应提示错误 (需求 2.1.3)。
- [x] **TC-017: 编辑/删除分组**: 重命名分组或删除空分组。
- [ ] **TC-018: 拖拽/移动**: (若支持) 或通过编辑页将密码移动到不同分组。

## 4. 搜索与过滤 (Search & Filter)

- [x] **TC-019: 实时搜索**: 输入字符时列表实时更新。
- [x] **TC-020: 多字段搜索**: 输入 URL 或 用户名 也能搜到对应记录 (需求 2.1.3)。
- [x] **TC-021: 按分组筛选**: 点击左侧分组，右侧仅显示该组内容。

## 5. 便笺模块 (Secure Notes)

- [x] **TC-022: 模块切换**: 点击顶栏“便笺”或快捷键 `Cmd+2` 切换 (需求 2.1.7)。
- [x] **TC-023: 新建便笺**: 创建自由文本便笺，支持独立分组。
- [ ] **TC-024: 置顶与归档**: 设置便笺为“置顶”或“归档”，验证排序变化 (需求 2.1.6)。
- [x] **TC-025: 便笺搜索**: 在便笺模块内搜索内容 (需验证本地解密搜索)。

## 6. 设置与工具 (Settings & Tools)

- [x] **TC-026: 密码生成器**: 生成符合规则(长度, 字符集)的强密码 (需求 2.1.2)。
- [ ] **TC-027: 数据导出 (JSON)**: 导出所有数据为 JSON。
- [ ] **TC-028: 数据导出 (加密ZIP)**: 设置密码导出加密 ZIP 包。
- [ ] **TC-029: 自动导出设置**: 配置自动导出频率(每分/每日等)，验证是否触发 (需求 2.3.1)。
- [x] **TC-030: 快捷键**: 验证 `Cmd+1` (密码), `Cmd+2` (便笺), `Cmd+F` (搜索), `Cmd+N` (新建) (需求 2.1.7)。

## 7. 界面与体验 (UI/UX)

- [ ] **TC-031: 响应式**: 调整窗口大小，布局自适应。
- [ ] **TC-032: 中文显示**: 时间格式为 YYYY-MM-DD HH:mm:ss，界面文案均为中文。
